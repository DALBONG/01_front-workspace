CSS 기본 선택자

선택자란?

특정 html 요소를 선택하고자 할 때 사용하는 기능
해당 요소를 선택하여 원하는 "스타일"과 "기능"을 적용.

1. 모든(전체) 선택자 : *

현재 이 문서상의 모든 요소들을 다 선택하고자 할 때 사용

             *{스타일 속성:값; 스타일 속성:값;}

2. 태그 선택자 : 태그 명

현재 이 문서상의 태그들을 다 선택할 때 사용

        태그명{스타일 속성:값; 스타일 속성:값}

3. 아이디 선택자 : #id명

현재 문서상의 특정 html 요소 "하나"만을 선택하고자 할 때 사용
단, 해당 요소에 고유한 id값을 부여해야 함.

        #id명{스타일속성:값; 스타일속성:값;}

4.클래스 선택자 : .

해당 문서 내, 원하는 요소 "여러개"를 선택하고자 할 때 사용

.클래스명{스타일 속성 : 값;}
