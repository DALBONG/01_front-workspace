폼관련태그

form태그 내 submit버튼 클릭시 해당 form안에 작성된 사용자가 입력한 값들을 서버로 넘기면서 요청하는 역할 수행
- form action="" , method="" 속성사용

*action속성 : 해당 폼에 사용자가 입력한 값을 전달하면서 요청할 서버의 경로 제시
*method속성 : 요청 전송 방식을 지정하는 속성 (get: 기본값/post)
- get : 요청시 사용자가 입력한 값들이 url에 노출되는 방식
- post : 요청시 사용자가 입력한 값들이 url에 노출되지 않는 방식.
form내에 submit 버튼 클릭시 action 지정한 서버로 사용자가 입력한 값들이 넘어가면서 요청됨.
(사용자가 입력한 값들은 항상 key=value 세트로 전달)

라디오 버튼(남/녀, 지역)과 체크박스(다중)관련 태그

라디오 버튼(type="radio")

- 제시한 여러개의 값들 중 오로지 한 개만 선택해야 할 때
- 제출시 현재 선택된 라디오 버튼의 밸류값으로 넘어감
- 네임 값을 똑같이 해줘야 같은 그룹으로 읽고, 단일 선택 가능


성별 :



체크박스(type=checkbox)

- 제시한 여러개의 값들 중 여러개 선택할 경우
- 제출시 현재 선택된 체크박스의 밸류값으로 넘어감
