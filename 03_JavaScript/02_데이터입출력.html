2. 데이터를 입력받는 구문(변수에 기록)

1_ 변수 = [window.]confirm("질문내용");
2_ 변수 = [window.]prompt("질문내용");
3_ 변수 = 선택한 요소.속성(id, className 등...)
4_ 변수 = 선택한 input요소.value;


1_ window.confirm("질문내용")

confirm함수 호출 시 질문내용과 "확인/취소" 버튼이 알람창 발생
확인버튼 클릭시 true / 취소버튼 클릭시 false로 반환


window.prompt("질문내용");

prompt 호출시 "질문내용"과 입력할 수 있는 "텍스트 상자"와
"확인/취소"버튼이 보여지는 알람창 발생
- 확인버튼 클릭시 텍스트 상자에 입력되어 있는 값이 문자열로 반환
(아무리 숫자가 써있어도 문자열로 반환)
- 취소 버튼 클릭시 null을 반환
(글자입력을 했어도 취소를 누르면 무조건 null)

JS 반복문

반복문 표현법 
(for문) : 특정 횟수나 범위만큼 반복하는 문
 ㄴ for(초기식; 조건식; 증감식){
    - 초기식: 변수의 선언
    - 조건식: 반복을 수행할지 안할지 조건을 판단하는 구문
    - 증감식: 변수를 증가하거나 감소시키는 구문
  => 위 3개의 값을 가지고 반복 횟수를 결정 }
